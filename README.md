Excel-like-DataForm
一個運用原生 JavaScript 實作的互動式表格元件，具備類似 Excel 的操作體驗，包括鍵盤導航、編輯模式切換與資料儲存機制。

🚀專案亮點
零框架依賴：100% 原生 JavaScript (Vanilla JS)，不使用 React/Vue，展現純粹的 DOM 操控與效能優化實力。

高效虛擬渲染：手刻虛擬滾動 (Virtual Scrolling) 技術，透過監聽 scroll 事件即時計算可視區域，面對 50,000 格以上的大數據量依然保持極致順暢。

狀態驅動架構：實作有限狀態機 (FSM) 管理編輯（EDITING）與選取（ACTIVE）模式，確保資料與視圖精確同步。

流暢鍵盤導航：完整支持全鍵盤操作（方向鍵焦點移動、Enter 進入/儲存編輯、Escape 放棄修改並還原舊值）。

無縫輸入體驗：透過狀態機精準調配 input 焦點，支援使用者在選取模式下「直接打字」即觸發編輯（流暢支援中英文輸入與蓋版優化）。

🛠️專案架構
Plaintext
├── index.html          # 精簡的入口 HTML，維護分離的 Editor 獨立圖層
├── main.js             # 應用程式進入點，負責全域事件綁定與生命週期協調
├── DataTable.js        # 核心商業邏輯、資料狀態維護、狀態機定義
├── Renderer.js         # 虛擬滾動計算、動態 DOM 生成、浮動編輯器定位
└── utils.js            # 工具函數（邊界檢查、Excel 欄位英文字母轉換演算法）
💎未來擴充方向
國際化與在地化支援：預計導入 Intl API，依據語系自動動態轉換日期格式、貨幣符號與數字格式（呼應雲端多國語系平台需求）。

資料庫概念驗證 ：預計為欄位建立 Schema 型別定義，在儲存時進行資料型別與格式校驗。

進階試算表功能：規劃實作 Undo / Redo 機制、複製/貼上功能，以及基礎公式計算（如 =A1+B1）。
