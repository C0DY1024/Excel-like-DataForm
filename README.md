一個運用原生JavaScript實作的互動式表格元件，具備類似 Excel 的操作體驗，包括鍵盤導航、編輯模式切換與資料儲存機制。

功能解說:
支援鍵盤操作（上下左右移動）
Enter 切換「選取 / 編輯」模式
Esc 可退出編輯模式
點擊滑鼠可選取或進入編輯
編輯內容即時儲存至資料模型
清楚的狀態管理（active / editing）
表格支援動態生成

設計重點:
使用 class 封裝整體邏輯
狀態集中管理於 state 物件
將「資料（data）」與「DOM」分離
將 UI 更新與狀態切換拆分處理

未來擴充方向:
格子內容超出時隱藏(保持所有格子的大小一致)
Undo / Redo 機制
複製 / 貼上功能
公式計算（如 =A1+B1）
多選操作
React 版本重構
